## Python+opencv 车牌识别小程序

+ ############

+ 数字图像处理课程设计作业

+ ############

+ 环境：

  Python3+OpenCV

  使用pycharm开发

  使用tkinter搭建界面

+ ############

+ `tmp/`文件夹是数字图像处理过程

+ `chepai/`文件夹是车牌图片

+ `pic/`文件夹是程序界面图

+ ############

## 开发环境配置

> `pip install numpy`

> `pip install pillow`

> `pip install opencv-python`
